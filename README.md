# Telegraph Clone

[Post your own notes anonymously](http://b00bl1k-telegraph.herokuapp.com/).

# Run locally

## Install

```
pip install -r requirements.txt
```

## Run

```
$ python server.py
```

Navigate to [localhost:5000](http://localhost:5000)
